# About Ambassadorセクション追加 - 提案書

**作成日**: 2026年1月8日
**提案者**: アンバサダー Kikumi

---

## 📌 提案の概要

現在のサイトに「About Ambassador（アンバサダーについて）」セクションを新設することを提案します。

### 配置場所
- **Linksセクション**と**フッター**の間
- ナビゲーションには追加せず、自然な流れで読者が到達できる位置
- ヒーローセクションの「アンバサダー Kikumi」からジャンプできるリンクを設置

### 掲載内容
- 小さめの似顔絵（丸型）
- アンバサダーとしての想い・背景を伝えるテキスト（以下の4段落）

```
初めてGreen Beans Coffeeを訪れたのはオープン間もない頃。ウッド調の空間と焙煎したてのコーヒーの香りに包まれてここは特別な場所だと感じました。それから3年通っていますが訪れる度に新しい発見があります。

カフェなのにホテルのような空間とサービス。化粧室のアメニティにも女性への心遣いを感じます。それまでは家でコーヒーを飲むことはなかったのに今では自宅でも毎日飲むようになりました。

都内で友人と再会したり打ち合わせをする時にはいつもGreen Beans Coffeeを選んでいます。グルテンフリーのランチやスイーツが楽しめるのも魅力の一つです。静かにコーヒーを味わう人、わんこと一緒にくつろぐ人。コーヒー片手にパソコンで仕事をする人、穏やかな時間が流れる場所です。

これからもアンバサダーとしてこのお店の魅力を定期的に発信していきます。
```

---

## ✅ なぜこの施策が有効か

### 1. 読者視点でのメリット

#### 信頼性の向上
現状では「アンバサダー Kikumi」という表記だけで、読者は以下の疑問を持つ可能性があります：
- 「このアンバサダーってどんな人？」
- 「なぜこの人が紹介しているの？」
- 「信頼できる情報なの？」

→ **About Ambassadorセクションで背景を明示することで、これらの疑問が解消され、サイト全体への信頼感が向上します。**

#### 共感と親近感の醸成
- 「3年間通い続けている」という実体験の明示
- 「初めて訪れた時の感動」「グルテンフリー生活での発見」など、具体的なエピソード
- 似顔絵を添えることで人間味と親しみやすさがアップ

→ **読者が「この人の紹介なら信頼できる」と感じやすくなります。**

#### 自然な導線
- ヒーローセクションの「アンバサダー Kikumi」をクリックすると、このセクションへジャンプ
- 興味を持った読者が、すぐに詳細を知ることができる

---

### 2. SEO/AEO観点でのメリット

#### E-E-A-T（Experience, Expertise, Authoritativeness, Trust）の強化

Googleが重視する品質評価基準において、特に以下が向上します：

- **Experience（経験・体験）**: 「オープン当初から3年間通い続けた」という実体験の明示
- **Authoritativeness（権威性）**: 公式に「アンバサダー」として認定されている立場の説明
- **Trustworthiness（信頼性）**: 顔（似顔絵）と名前を出すことで透明性が向上

#### Answer Engine Optimization（AEO）の強化

AI検索エンジン（ChatGPT、Perplexity、Geminiなど）が情報を引用する際、以下の点が評価されます：

- **一次情報の明示**: 「実際に3年間通っているアンバサダーが書いている」ことが明確に
- **著者情報の透明性**: 「誰が」「なぜ」書いているかが明確
- **体験に基づいた信頼性**: AIが「実体験に基づく信頼できる情報」と判断しやすくなる

→ **Answer Engineが情報を引用する際、このサイトを優先的に選択する可能性が高まります。**

#### 構造化データ（JSON-LD Person Schema）の追加

技術的な施策として、アンバサダーの情報を構造化データとして記述します。これにより：

- **検索エンジンに「誰が書いたか」を明確に伝達**
- Google Knowledge Graphへの情報反映の可能性
- リッチリザルト（検索結果の拡張表示）の可能性向上
- AI検索エンジンが著者情報を正確に理解

## 🎨 デザイン案

### レイアウト
- **セクションタイトル**: 「About Ambassador」（他セクションと統一）
- **似顔絵**: センター上部に小さめ配置（120-140px、丸型）
- **テキスト**: センター寄せ、最大幅制限で読みやすく
