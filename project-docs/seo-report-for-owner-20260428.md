## 1. 今回実施した作業：画像の説明文（alt属性）の拡充

### どんな作業をしたか

サイト内の全18枚の画像に、内容を説明する「隠れテキスト」を設定しました。

この「隠れテキスト」（alt属性）は、画面上には表示されませんが、  
**GoogleなどのAIが画像を読み取るときに参照する説明文**です。

**改善前の例：**
```
ペット向けサービス（9文字）
```

**改善後の例：**
```
Green Beans Coffeeが用意するペット向けサービス。お水・水飲みボウルの提供やチェキ撮影など、
愛犬連れに嬉しい北参道・千駄ヶ谷エリアのカフェのサービス（74文字）
```

---

### なぜこの作業が重要なのか

「北参道でペット同伴OKのカフェを教えて」とAIや検索エンジンに質問したとき、  
Googleは**画像の説明文も含めてページ全体を読んで**、情報を判断します。

説明文が短いと「このページに北参道やペット同伴に関する情報がある」と  
正確に伝わらず、検索結果に表示されにくくなります。

---

### 期待できる効果

**① Google画像検索からの流入**  
「北参道 犬同伴カフェ」「ダガヤサンドウ ペットフレンドリー」などで  
画像検索した人に表示される可能性が高まります。

**② AIへの回答精度向上**  
ChatGPT・Gemini・Perplexityなどが「北参道でペット同伴OKのカフェは？」と聞かれた際に、  
このサイトを情報源として引用しやすくなります。

**③ アクセシビリティの向上**  
目が不自由な方が使う読み上げソフトが、画像の内容を正確に伝えられるようになります。

**④ サイト全体の信頼性向上**  
検索エンジンは「丁寧に作られたサイト」を高く評価します。  
短い説明文のまま放置するより、充実させた方がサイト全体の評価が上がります。

---

### 効果が出るまでの時間

Googleがサイトを再クロール（確認）するまで、数日〜2週間程度かかります。  
すぐには変化が現れませんが、中長期的に効いてきます。

## 2. 次の改善：店舗の代表写真の指定

### 何が変わるか

現在、Google検索やGoogleマップで店舗情報と一緒に表示される写真は、  
**Googleが自動で選んでいます**。

代表写真を設定すると、**「この写真を使ってください」とGoogleに指定できる**ようになります。  
一度設定すれば更新は不要です。

### お願いしたいこと

すでにサイトに掲載されている写真の中から、**お店の雰囲気が一番よく伝わる写真を1〜3枚**選んでください。

ブラウザでサイトを開き、使いたい写真の上で右クリック →「画像のアドレスをコピー」でURLが取得できます。  
URLを教えていただければ、こちらで設定します。
