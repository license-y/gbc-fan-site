# ペット撮影会 素材収集ガイド

Green Beans Coffeeペット特化サイト用の写真・動画素材を効率的に収集するためのガイドです。

---

## 1. 撮影の基本ルール

### スマホの向き

| 推奨 | 用途 |
|------|------|
| **横向き（ランドスケープ）** | Webサイト用（Hero画像、セクション背景、カード型レイアウト） |
| 縦向き（ポートレート） | SNS用（サイトには使用しない） |

**スタッフへの指示例:**
> 「スマホを横に持って撮影してください。特にお店の雰囲気がわかる引きの写真が欲しいです」

### 推奨アスペクト比

| 用途 | アスペクト比 | 理由 |
|------|-------------|------|
| Hero画像（メインビジュアル） | **16:9 横向き** | PC表示で最も映える |
| セクション背景 | **16:9 横向き** | 全幅表示に最適 |
| カード型レイアウト | **4:3 横向き** | グリッド表示に適合 |
| Visit Diary | **4:3 または 1:1** | 柔軟に使える |

## 2. 欲しい写真リスト

### 必須（Hero・メインビジュアル用）

| 優先度 | 内容 | 撮影のコツ |
|--------|------|-----------|
| 1 | **ペットとオーナーがくつろぐ様子**（テラス席で） | オーナーの笑顔とペットの自然な姿、背景に店舗が映るように |
| 2 | **店内全景 + ペット** | お店の雰囲気が伝わる引きの画、ペットは中央～手前に |
| 3 | **チェキブースでの撮影シーン** | ブース全体と撮影中の様子 |

### あると良い（各セクション用）

| 優先度 | 内容 | 撮影のコツ |
|--------|------|-----------|
| 4 | **犬用の水やアメニティ** | クローズアップで設備がわかるように |
| 5 | **スタッフがペットと触れ合う様子** | 温かい雰囲気が伝わる自然な表情 |
| 6 | **複数組のペット連れ来店者** | 賑わい感、奥行きのある構図 |
| 7 | **コーヒーとペットが映るテーブルショット** | コーヒーカップとペットの両方にピントが合うように |

### 追加コンテンツ用（チェキブース・参加者の声セクション）

| 優先度 | 内容 | 撮影のコツ |
|--------|------|-----------|
| 8 | **チェキブース詳細** | ブースの全景、飾り付け、装飾 |
| 9 | **過去のチェキ写真が飾られている壁** | 壁全体を俯瞰で |
| 10 | **参加者の声カード** | 複数枚を並べて俯瞰撮影、個別のカードも数枚 |

## 3. 動画について

| 用途 | 形式 | 推奨 |
|------|------|------|
| サイト背景動画（将来的に） | **横向き 16:9** | 10-15秒のループ用 |
| SNS・リール用 | 縦9:16 | サイトには使わない |

**今回のおすすめ:**
- 動画は横向きで2-3本撮っておくと将来使える
- 静止画を優先し、動画は余裕があれば

**動画の内容例:**
- ペットがテラス席でくつろぐ様子（10秒程度）
- チェキブースでの撮影風景（15秒程度）
- 店内を歩くペットと飼い主（10秒程度）

## 4. 参加者の声カード収集

### カードのテンプレート案

撮影会当日、参加者に記入してもらうカードの項目:

```
┌─────────────────────────────────┐
│  ペット撮影会 参加者カード     │
├─────────────────────────────────┤
│                                 │
│  ペットの名前:                  │
│  ─────────────────────────      │
│                                 │
│  今日の感想:                    │
│  ─────────────────────────      │
│  ─────────────────────────      │
│                                 │
│  GBCの好きなところ:             │
│  ─────────────────────────      │
│                                 │
│  □ サイトへの掲載OK             │
│                                 │
└─────────────────────────────────┘
```

### カードの撮影方法

1. **複数枚を並べて俯瞰撮影**
   - 4-6枚を整列させて上から撮影
   - 背景は木目調やシンプルな色

2. **壁に飾ってある状態**
   - チェキと一緒に飾られている雰囲気写真

3. **個別のカード**
   - 特に良い感想が書かれたものを数枚

## 5. 撮影チェックリスト

### 撮影前の準備

- [ ] スマホのストレージ空き容量確認
- [ ] カメラアプリの設定確認（HDRオン推奨）
- [ ] 参加者カードの準備
- [ ] 掲載許可の確認方法を決定

### 撮影当日

**必須ショット:**
- [ ] ペットとオーナーがくつろぐ様子（横向き）
- [ ] 店内全景 + ペット（横向き）
- [ ] チェキブースでの撮影シーン（横向き）

**あると良いショット:**
- [ ] 犬用の水やアメニティ（クローズアップ）

**チェキブース・参加者の声用:**
- [ ] チェキブース全景
- [ ] 過去のチェキ写真が飾られている壁

- [ ] ペットがくつろぐ様子（横向き、10秒）

## 6. 画像の整理と命名

撮影後、以下のルールで整理してください:

### フォルダ構成

```
撮影会_YYYYMMDD/
├── hero/           ← メインビジュアル用
├── pet-friendly/   ← Pet Friendlyセクション用
├── cheki-booth/    ← チェキブースセクション用
├── visit-diary/    ← Visit Diary用
└── raw/            ← 未整理・選定前
```

### 命名規則

```
hero_pet-and-owner_01.jpg
pet-friendly_water-bowl_01.jpg
cheki-booth_wall_01.jpg
visit-diary_pomeponkun_01.jpg
```

## 7. 画像の最適化

サイトにアップロードする前に、以下のサービスで最適化してください:

- **TinyPNG** (https://tinypng.com/): PNG・JPEG最適化の定番
- **Squoosh** (https://squoosh.app/): Googleが提供する画像圧縮ツール
- **Optimizilla** (https://imagecompressor.com/): 複数画像の一括処理対応

**推奨設定:**
- 幅: 1920px以下（Hero用は1920px、その他は1200px程度）
- 品質: 80%程度
- フォーマット: JPEG（写真）、WebP（対応可能なら）
