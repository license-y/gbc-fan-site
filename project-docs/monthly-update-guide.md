# 月次アップデート 作業ガイド

## 全体の流れ

| ステップ | 作業 | 自動化 |
|---------|------|--------|
| 1 | Canva からファイルをダウンロード | 手作業 |
| 2 | inbox/ にファイルを配置 | 手作業 |
| 3 | スクリプト実行（圧縮・移動・ローテーション） | **自動** |
| 4 | index.html のテキスト編集（豆の名前など） | 手作業 |
| 5 | ブラウザで確認 | 手作業 |
| 6 | コミット＆プッシュ | 手作業 |

---

## 各コンテンツの手順

### 焙煎体験 / ハンドドリップ体験イベントのチラシ

**Canva での作業:**
1. 「体験イベント」フォルダ内の該当デザインを開く
2. JPG でダウンロード（0.75倍 / 1,061×1,500px / 200KB制限）

**inbox への配置（ファイル名を変えること）:**

```
inbox/
├── event-roasting.jpg     ← 焙煎体験チラシ
└── event-handdrip.jpg     ← ハンドドリップ体験チラシ
```

---

### Monthly 豆のチラシ

**Canva での作業:**
1. 「豆バリスタ説明＆Monthly豆」フォルダ内の該当デザインを開く
2. JPG でダウンロード（ファイル名はそのままで OK）

**inbox への配置:**

```
inbox/
└── monthly-coffee/
    ├── 1.jpg     ← 豆チラシ 1枚目
    └── 2.jpg     ← 豆チラシ 2枚目
```

**スクリプト実行後の追加作業:**
- `index.html` の豆の名前テキストを最新に書き換える

---

### GBC マガジン

**PDF の準備（スクリプト実行前の手作業）:**
1. PDF を開き、印刷設定で「2ページを1枚・見開き」表示の PDF として保存
2. Adobe Acrobat などで PDF → JPG に変換
   - 変換サービス: https://www.adobe.com/jp/acrobat/online/pdf-to-jpg.html

**inbox への配置（ファイル名を変えること）:**

```
inbox/
└── magazine-latest.jpg     ← 今月号の画像
```

**スクリプトが自動で行うこと:**
1. 古い `back-06.jpg`（7ヶ月前）を削除
2. 既存ファイルを番号繰り上げ（back-05→06, back-04→05 … back-01→02）
3. 現在の `latest.jpg` → `back-01.jpg` に移動
4. inbox の新しいファイルを圧縮して `latest.jpg` として配置

---

## スクリプトの実行方法

### 1. inbox にファイルを配置する

Finder でプロジェクトフォルダを開き、`inbox/` フォルダにファイルをドラッグ＆ドロップ。

> **ポイント:** 更新するコンテンツのファイルだけ置けば OK。全部そろっていなくても、あるファイルだけ処理されます。

### 2. ターミナルでスクリプトを実行する

Claude Code のターミナル（または通常のターミナル）でプロジェクトフォルダに移動し:

```
npm run update-images
```

## トラブルシューティング

### 「inbox/ に処理するファイルが見つかりません」と表示された

inbox/ にファイルが置かれていない状態です。ファイルを配置してから再実行してください。

### 「TinyPNG API キーが未設定です」と表示された

`.env.local` に `TINYPNG_API_KEY` が設定されていません。設定すると自動圧縮が有効になります。設定がない場合も**圧縮なしでファイルの移動は行われます**。

### 「TinyPNG API エラー」と表示された

- APIキーが間違っている可能性があります
- 月の使用回数（500回）を超えた可能性があります（通常の月次更新では起こらない）
- インターネット接続を確認してください

この場合も**圧縮なしでファイルの移動は行われます**。

---

## TinyPNG API キーについて

- 取得先: https://tinypng.com/developers
- 無料枠: **月500回**（月次更新で使う枚数は最大5枚なので十分）
- 設定場所: `.env.local` ファイルの `TINYPNG_API_KEY=` に記載
- 使用回数はスクリプト実行後に表示されます
