## 使い方
1. 各画像の「★ 推奨alt属性（編集してください）」の部分を自由に編集してください
2. 必要に応じて追加のメモを書き込んでください
3. 編集完了後、Claude Codeに伝えてください
4. 慎重にHTMLへ反映します

## 改善のポイント

### 追加したキーワード
- **店名**: Green Beans Coffee（全画像に統一）
- **地域**: 北参道、千駄ヶ谷、ダガヤサンドウ（適切な箇所に）
- **特徴**: 自家焙煎、ペットフレンドリー、グルテンフリー、スペシャルティコーヒー
- **業態**: カフェ、コーヒー専門店、ワークショップ

### SEO/AEO効果
1. **Answer Engine対応**: より具体的な情報で質問に答えやすく
2. **地域SEO**: 北参道、千駄ヶ谷を明記して地域検索に強化
3. **特徴訴求**: ペット同伴、グルテンフリーなどのニッチキーワード
4. **自然な日本語**: AIが引用しやすい文章構造
